# Contributing

Thank you for your interest in contributing to this starter kit!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/convex-nextjs-polar-clerk-starter-kit.git`
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

```bash
# Run both Next.js and Convex dev servers
pnpm dev:all

# Or run them separately
pnpm dev          # Next.js dev server
npx convex dev    # Convex dev server
```

## Code Style

- Use TypeScript for all new code
- Follow existing patterns in the codebase
- Run `pnpm lint` before committing
- Use meaningful commit messages

## Pull Requests

1. Ensure your code passes linting: `pnpm lint`
2. Ensure the build succeeds: `pnpm build`
3. Update documentation if needed
4. Submit a pull request with a clear description of changes

## Reporting Issues

When reporting issues, please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (Node version, OS, etc.)

## Questions?

Feel free to open an issue for any questions or discussions.
